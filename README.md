## 🚀 Projects I've Built

**🏫 WhatsApp-Based School Attendance Automation**
A production Flask app where teachers mark only absentees, parents get instant WhatsApp alerts, and daily/monthly PDF reports generate automatically. Built with role-based dashboards (Clerk auth), PostgreSQL, and a swappable WhatsApp adapter architecture, deployed on Railway.

**🔗 Criminal Evidence Management System (Blockchain)**
A tamper-proof digital evidence platform using cryptographic hashing, immutable ledgers, and RBAC for secure, auditable access to sensitive records.

**🛡️ Malware Detection using Machine Learning**
A TensorFlow neural network for malware classification, with feature engineering and tuning to reduce false positives.
